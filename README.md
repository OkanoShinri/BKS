# BKS

ブロック崩しシューティング

- src : ソースコード
- bin : 実行ファイル
- README : これ

## 概要
ブロックくずし要素を取り入れた弾幕風弾避けゲームです。
黄色いボールが画面内を飛び回り、ぶつかったブロックを破裂させます。
黄色いボールには当たっても大丈夫ですが、ブロックから飛び散る弾に当たると1ミスです。
